# ng-ssl

Certificates are generated by certbot at runtime.

Environement variables:
    CERTBOT_DOMAIN
    CERTBOT_MAIL


## Quick start

```
docker build -t ng-ssl .
docker run -d --rm -p 80:80 -p 443:443 \
  --network host \
  -e CERTBOT_DOMAIN=mydomain.net \
  -e CERTBOT_MAIL=admin@mydomain.net \
  ng-ssl
```
